# The Graph - GraphQL

[The Graph](https://thegraph.com) lets you query on chain data with GraphQL.

We provide a Subgraph for the BURN contract, allowing you to query data beyond what our REST API offers,
such as seeing the timestamp of when each wallet was burned.
